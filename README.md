# Stock Options Calculator

A comprehensive web application for calculating and visualizing stock option values, vesting schedules, RSUs, and exit scenarios.

## Features

### Existing Features (Fully Implemented)
1. **Quick Calculator** - Calculate current option values with scenario planning
2. **Vesting Timeline** - Visualize vesting schedules over time with cliff periods
3. **RSU Calculator** - Calculate Restricted Stock Unit values and tax implications

### New Features (Ready for Implementation)
4. **Portfolio Mode** - Track multiple option grants with aggregate statistics
5. **Exit Scenarios** - Model IPO, acquisition, and secondary sale outcomes
6. **Data Persistence** - Auto-save to localStorage
7. **Export Features** - Export to CSV and print/PDF

## Current Status

**Working Version:** The application currently works as a single-file `index.html` with all CSS and JavaScript embedded.

**Modular Version:** CSS has been extracted to `styles.css`. JavaScript modularization is in progress.

## File Structure

- `index.html` - Current working single-file version
- `styles.css` - Modular CSS (complete and ready)
- `app.js` - Modular JavaScript (in progress)
- `index.backup.html` - Backup of working version

## Quick Start

Simply open `index.html` in a web browser or deploy to GitHub Pages.

## Technologies

- HTML5, CSS3, JavaScript (ES6+)
- Chart.js 4.4.1
- LocalStorage API
